1.2 Grundlagen der Kombinatorik und Zahlentheorie

a)

Permutation:
Eine beliebige Abbildung einer endlichen Menge nennt man Permutation ohne Wiederholung, solange die Abbildung bijektiv ist, das heißt jedem Element eine eindeutige Position zugeordnet wird.
Eine Permutation mit Wiederholung ist eine Permutation bei der Elemente mehrfach vorkommen können.

Variation:
Eine Variation ist eine Abbildung einer Menge bei der die Reihenfolge berücksichtigt wird, aber nicht jedem Element eine Position zugeordnet wird. Bei Variationen mit Wiederholung dürfen Elemente mehrfach vorkommen.

Kombination:
Eine Kombination ist eine Abbildung einer Menge, bei der die Reihenfolge nicht berücksichtigt wird. Bei Kombinationen mit Wiederholung dürfen Elemente in der Menge mehrfach vorkommen.
